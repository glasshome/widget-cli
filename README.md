# @glasshome/widget-cli

CLI for creating and managing GlassHome widgets.

## Install

```bash
bun add -g @glasshome/widget-cli
```

## Quick Start

```bash
bunx @glasshome/widget-cli create my-widget
cd my-widget
bun install
bun run dev
```

## Commands

| Command | Description |
|---------|-------------|
| `create <name>` | Scaffold a new widget project |
| `validate` | Validate widget manifest and bundle |
| `info` | Show widget metadata |
| `publish` | Publish widget to the registry |
